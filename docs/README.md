## 🚀 快速导航

如果你是第一次使用本工具，建议按以下顺序阅读文档：

1. **快速开始**: 返回根目录查看 [QUICK_START.md](../QUICK_START.md) - 5分钟快速上手
2. **完整说明**: 查看 [README.md](../README.md) - 了解所有功能特性
3. **项目结构**: 阅读 [PROJECT_STRUCTURE.md](./PROJECT_STRUCTURE.md) - 了解代码组织
4. **使用示例**: 参考 [EXAMPLES.md](./EXAMPLES.md) - 学习具体用法
5. **待处理任务**: 查看 [PENDING_TASKS_GUIDE.md](./PENDING_TASKS_GUIDE.md) - 了解如何处理转换失败的项目
6. **多语言生成**: 参考 [TRANSLATE_GUIDE.md](./TRANSLATE_GUIDE.md) - 学习如何生成多语言配置
7. **变更日志**: 查看 [CHANGELOG.md](../CHANGELOG.md) - 了解最新更新
